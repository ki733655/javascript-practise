# JavaScript - learn by building

This repository is a collection of my JavaScript learning journey — not just learning concepts theoretically, but actually implementing them practically through coding, problem-solving, and mini experiments.

I believe the best way to truly understand JavaScript is by writing code consistently, making mistakes, debugging them, and building logic from scratch. This repo reflects that process.

## What You'll Find Here

- JavaScript fundamentals
- Arrays & Objects
- Functions & Closures
- Async JavaScript
- Promises & Async/Await
- DOM Manipulation
- Event Loop Concepts
- Small practical implementations

## Goal

To become highly confident in JavaScript by:
- Writing code daily
- Understanding concepts deeply
- Building strong problem-solving skills
- Learning how JavaScript actually works behind the scenes

## Progress Over Perfection

I'm continuously updating this repository as I learn new concepts and improve my understanding.
